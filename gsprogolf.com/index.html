    • What are the minimum and recommended system requirements?

      GSPro requires a Windows PC (Windows 10 or Windows 11)


      Recommended minimum hardware for 1080p play:

      1. 10GB free space
      2. GTX 3060
      3. 16GB memory
      4. Stable internet Connection
      5. Ethernet port and/or BT depending on Launch Monitor

      Recommended hardware for a great 1080p experience:

      1. 20GB free space
      2. RTX 3070/3060ti GPU
      3. 16GB memory

      Recommended hardware for a great 4k experience:

      1. 20GB free space
      2. RTX 3080 (or better) GPU
      3. 32GB memory
    • What happened to the lifetime add-on option?

      The GSPro Lifetime Add-on is no longer available. As course building tools, the user created courses themselves, online dependencies, launch monitor updates, and new features continue to evolve, it has become increasingly difficult to support outdated versions of GSPro. To maintain our focus and commitment to constantly improving GSPro we now have the single, unchanged price, yearly subscription option.

      Does this impact existing lifetime users? No, your license will continue to work as before.
